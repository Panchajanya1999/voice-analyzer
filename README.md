# AI Voice Detector 🎙️🤖

A real-time voice analysis system that detects whether speech is generated by humans or AI. The system captures 5 seconds of audio and uses machine learning to classify the voice source with confidence scoring.

## Features ✨

- **Real-time Detection**: Records and analyzes 5-second voice samples
- **High Accuracy**: Uses 40+ audio features for robust classification
- **Confidence Scoring**: Provides reliability metrics for each prediction
- **Easy Training**: Simple interface for training with custom datasets
- **Continuous Monitoring**: Option for ongoing voice analysis
- **Model Persistence**: Save and load trained models

## How It Works 🔧

The system analyzes multiple audio characteristics:

- **Spectral Features**: Frequency distribution patterns
- **MFCC Analysis**: Mel-frequency cepstral coefficients
- **Temporal Patterns**: Rhythm and timing variations
- **Pitch Analysis**: Fundamental frequency characteristics
- **Harmonic Content**: Natural vocal tract resonances
- **Periodicity**: Regularity in speech patterns

## Quick Start 🏃‍♂️

### 1. Basic Usage

```python
from voice_classifier import AIVoiceDetector
from main import RealTimeVoiceDetector

# Initialize the detector
detector = RealTimeVoiceDetector()

# Record and analyze voice
result, confidence = detector.detect_voice_type()
print(f"Result: {result}, Confidence: {confidence:.2%}")
```

### 2. Command Line Interface

```bash
python main.py
```

Follow the interactive menu:
- `1` - Record and detect single voice sample
- `2` - Continuous monitoring mode
- `3` - Train new model
- `4` - Exit

### 3. Training Your Own Model

```python
from voice_classifier import AIVoiceDetector

# Initialize detector
detector = AIVoiceDetector()

# Prepare training data
human_voices = ["human1.wav", "human2.wav", "human3.wav"]
ai_voices = ["ai1.wav", "ai2.wav", "ai3.wav"]

# Train model
detector.train_model(human_voices, ai_voices)

# Save model
detector.save_model("my_voice_model.pkl")
```

## Training Data Requirements 📊

For optimal performance, collect:

### Human Voices
- **Quantity**: 20-50 samples per person
- **Duration**: 5-10 seconds each
- **Variety**: Different ages, genders, accents
- **Quality**: Clear recording, minimal background noise
- **Format**: WAV files, 16kHz+ sample rate

### AI Voices
- **Sources**: Multiple TTS systems (Google, Amazon, OpenAI, etc.)
- **Types**: Different AI models and voice styles
- **Quantity**: Match human sample count
- **Diversity**: Various synthetic voice characteristics

## Performance Metrics 📈

Expected accuracy with good training data:
- **Overall Accuracy**: 85-95%
- **Human Voice Detection**: 90-95%
- **AI Voice Detection**: 80-90%
- **Processing Time**: <2 seconds per 5-second sample

## Troubleshooting 🔍

### Common Issues

**1. Audio Recording Problems**
```bash
# Linux: Check audio permissions
sudo usermod -a -G audio $USER

# Test microphone
python -c "import sounddevice as sd; print(sd.query_devices())"
```

**2. Model Training Errors**
- Ensure audio files are in correct format (WAV preferred)
- Check file paths are correct
- Verify sufficient training samples (minimum 10 per class)

**3. Low Accuracy**
- Increase training data diversity
- Ensure balanced dataset (equal human/AI samples)
- Check audio quality and consistency
